API Documentation Of Fandom WebApp

Authentication

Register User

POST /api/user/register

To register a new user.

Login User

POST /api/user/login

To login the user if the credentials are correct.

Logout User

POST /api/user/logout

To logout from the current account.

Delete User

DELETE /api/user/:id

To delete your account if the ID matches.

Media Data

Get All Movies

GET /api/details/movies

To get all the movies.

Get All TV Series

GET /api/details/tvseries

To get all the TV series.

Get Trending Data

GET /api/details/trending

To get all the trending related data (movie or TV series).

Get Popular Data

GET /api/details/popular

To get all the popular related data (movie or TV series).

Get Individual Details

GET /api/details/:id/:mediatype

To get individual details of a movie or TV series if the ID and mediatype match (mediatype = movie || tv).
